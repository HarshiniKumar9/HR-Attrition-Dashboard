# 📊 HR Attrition Analysis Dashboard (Power BI)

This Power BI project analyzes employee attrition trends and workforce patterns to help understand the key factors affecting employee turnover.

## 🔍 Dashboard Overview

* **Total Employees:** 1470
* **Attrition Rate:** 16%
* **Average Monthly Income:** 6503

## 🧠 Key Insights

* Higher attrition was observed in the **Research & Development** and **Sales** departments.
* Employees in the **lower salary band** showed higher attrition.
* Attrition was more noticeable during the **early years of service**.
* The **30–40 age group** showed significant attrition levels.
* Male employees had a higher attrition count compared to female employees.
